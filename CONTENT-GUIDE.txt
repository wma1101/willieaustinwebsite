======================================

on the site. Open the HTML file in any text editor (TextEdit, VS Code,
Sublime, etc.) and change the words between the > and < symbols.

Tip: Use Cmd+F to search for the current text, then replace it.

HOW TO ADD A NEW ARTICLE
=====================================================================

1. Duplicate editorial.html and rename it (e.g. "my-new-article.html")

2. Create an image folder:
   assets/images/editorials/my-new-article/

3. Add your images to that folder:
   hero.jpg, detail-wide.jpg, full-bleed.jpg, etc.

4. Open the new HTML file and change:
   - The <title> tag in <head>
   - Hero category, title, author, date
   - All article body text
   - Image src paths to point to your new folder
   - Image captions and alt text

5. Add a link to it from index.html (copy one of the stream blocks
   and change the headline, excerpt, image, and href)

6. Add a link in the nav if you want it in the menu


=====================================================================
HOW TO ADD A NEW COLLECTION
=====================================================================

1. Duplicate collection.html and rename it

2. Create an image folder:
   assets/images/collections/my-collection/

3. Add look images: hero.jpg, look-01.jpg, look-02.jpg, etc.

4. Edit the HTML: title, season, description, designer notes, credits

5. Link to it from index.html
